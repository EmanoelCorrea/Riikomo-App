1. Sobre o Projeto:
O projeto tem como objetivo facilitar o aprendizado da Linguagem Portuguesa para 
as séries iniciais do Ensino Fundamental(índigenas).
----------------------------------------------------------------------------------
2. Criação do App:
Para a criação do App vamos utilizar o React Native + Expo. 
E para criar o tamplate vamos utilizar os seguintes comandos:
    Comando: "npx create-expo-app --template"
    Opção usada: Navigation(TypeScript).

    2.1 Pasta do Projeto:
        .vscode = Não mexer.
        assets = Não mexer, somente se souber.
        src <- Principal para a aplicação.
           /app
           /components
           /constants
        node_modules = Não mexer.
        restante não mexer.

    2.2 Solução de Ambiente:
        Comando: "npm i" ou "npm install"
    
    2.3 Para rodar o projeto:
        Comando: "npx expo start"
----------------------------------------------------------------------------------
3. Para versionamento utilizaremos o GitHub:
   3.1 Branch(ramo):
       Lado esquerdo inferior: aonde está inscrito "main", vamos clicar e adicionar
       uma nova branch:
       Apos feito a alteração no Projeto, vá no icone de ramificação ou aperte 
       (ctrl + Shift + G).
       Agora em "message" você colocará algo referente a mudança no codigo.
       E em seguida fazer o "commit" e clicar em sim no restante das mensagens no
       lado direto da tela.
